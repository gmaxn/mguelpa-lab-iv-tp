Sobre el trabajo practico: TP-Sala-de-Juegos.

El punto final de este trabajo practico consitia en desarrollar un juego propio o existente distinto de los que se encontraban dentro de las consignas de este trabajo práctico. El Juego a elegido a desarrollar fue el Tetris, cuyas funciones son, ademas de la logica del motor del juego, registrar el puntaje del jugador a medida que logra completar el objetivo. El juego se encuentra desarrolado completamente en java script utilizando el tag HTML "canvas" y una serie de matrises las cuales se iran definiendo aleatoreamente.
